AI Agents — myVNG · Design review
×
Flow 1 — Vào chat với Agent
1
Nhấn tab AI Agents (tab 2 trên bottom bar) → vào thẳng chat với Chat Default Agent (được config ở trang admin)
2
Nhấn < back góc trái header → về màn trước đó
3
Nhấn ⋮ ba chấm trên header → menu: New chat · Conversations (mở history drawer)
Flow 2 — Gửi tin nhắn & demo UpRace
1
Nhấn vào ô chat input → tự điền "thông tin sự kiện uprace 2025" → nhấn ↑ để gửi → nhận response với text, 2 media grid (ảnh/video), và 3 sections (File đính kèm · Hỗ trợ · Nguồn tham khảo) mặc định collapse — nhấn để mở từng section. Header sẽ hiện conversation title sau khi gửi tin đầu tiên
2
Nhấn topic chip bất kỳ → prompt list hiện lên → nhấn 1 prompt → text điền vào input → gửi để xem response thông thường
3
Nhấn + → attach sheet: Camera / Photos / Files (nhấn để thêm file, tối đa 5) + Web search tick + chọn Agent
4
Sau khi gửi file, nhấn icon 📁 trên header → Files panel mở bên phải hiện danh sách file đã gửi
Flow 3 — Switch agent & pin
1
Nhấn + → chọn Agent khác → new chat với agent đó, icon + tên + mô tả thay đổi
2
Nhấn ⋮ ba chấm → chọn Conversations → drawer trượt ra → ··· trên History → Pin / Rename / Delete · ··· trên Pinned → Unpin
3
Action bar dưới response: ↓ lưu · copy · 👍 · 👎 — like/dislike toggle, mutual exclusive
💡 Các tính năng chưa được mock — dựa vào behavior hiện tại anh QuynhND3 đã build ở UAT
• Prompt gallery (icon ✦ và "Xem thêm")
• Rename conversation
• Delete conversation
• Camera / Photos trong attach sheet
• Search conversations trong History drawer
• New chat từ overflow menu (⋮)
• Nhấn vào conversation trong History drawer để load lại chat
