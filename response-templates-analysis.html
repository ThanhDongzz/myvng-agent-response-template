Files đính kèm
Vấn đề gốc: Hiện tại hoàn toàn chưa có — crawl pipeline chưa lưu metadata files, agent không biết bài viết có file đính kèm, response không show gì cả. Đây là gap cần fill từ BE trước khi FE có thể làm gì.

Lưu ý: BE phải update crawl pipeline để lưu file metadata (name, type, size, url) trước — đây là prerequisite bắt buộc cho tất cả options.

Business Case 02
IT Support Q&A
Agent: Helpdesk Agent OpenAI · RAG thuần từ KB
Đặc điểm: Đây là RAG thuần — agent chỉ đọc KB và trả lời, không gọi tool, không cần action từ user. Response dạng text markdown (bold, bullet list) — phù hợp với loại nội dung này. Không cần build tool hay thay đổi gì thêm ở phase này.
✓ Không cần thay đổi response template
Chất lượng câu trả lời phụ thuộc trực tiếp vào độ đầy đủ của KB. Các cải tiến về nguồn tham khảo, files, contacts sẽ apply tự động khi BC00 hoàn thành.
Action cần làm Lên plan push các team leader trong IT chủ động viết docs và cung cấp cho KB — để agent có thể trả lời được các câu hỏi đặc thù của từng team, không chỉ thông tin chung chung. BA/PO phối hợp với IT leads để thực hiện.
